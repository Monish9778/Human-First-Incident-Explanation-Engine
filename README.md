# Human-First-Incident-Explanation-Engine


A simple web-based tool that converts technical error messages into
human-readable explanations with clear next steps.

## Features
- Paste any technical error message
- Get a clear explanation:
  - What happened
  - Why it happened
  - How to fix it
- Beginner-friendly explanations

## Tech Stack
- Python
- Flask
- HTML/CSS

## How to Run
```bash
pip install -r requirements.txt
python app.py
